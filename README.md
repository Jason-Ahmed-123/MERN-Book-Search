# MERN-Book-Search

## Purpose
A book search engine built using the MERN stack, with a React front end, MongoDB database, and Node.js/Express.js server and GraphQL API.

## Built With
* JavaScript
* React
* MongoDB
* Node.js and Express.js
* GraphQL
* Apollo Server
